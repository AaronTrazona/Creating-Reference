
Creating a Library/Reference in C#

1.) In Start Page Microsoft Visual Studio >  New  > Project >  Windows > Look for Class Library Project
2.) Create a Class then Create any method or properties.
3.) You are finish build the project then find the .dll of the project you make.
4.) Right Click the project > Open Folder in Windows Explorer > Bin > Debug > you can see your .dll copy that .dll to your other project.
5.) In your other project > Right Click the Reference > Add Reference > Browse the Library you make so that you can access the method you make in the library..

THAT'S ALL ...
